# LoGeek
LoGiC.NET Deobfuscator, supports all protections except proxy calls

# Supported Protections
<pre>
- Junk Methods
- Encoded Strings
- Int Obfuscation
- Remove AntiDe4dot
- Remove Watermark
- Remove AntiTamper Protection
- Module Renaming

Not Supported:
- Proxy Calls
</pre>

# Usage
Drag & Drop to Clean
